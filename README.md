# MBERE ML

## Layout

- `data/` for raw, processed, and external datasets
- `notebooks/` for exploration, feature engineering, training, and evaluation work
- `ml/` for reusable machine learning code, artifacts, and tests
- `backend/` for API and service code
- `frontend/` for the user interface
- `docs/` for proposals, diagrams, architecture notes, and API docs
- `scripts/` for automation and helper scripts
